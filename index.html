Exam Domains Quick Reference

1. Author & Maintain Workflows (40%)
  • Triggers: push, pull_request, schedule, workflow_dispatch
  • YAML: jobs, steps, matrix, needs, if conditionals
  • Secrets, GITHUB_TOKEN, env vars, caching, artifacts
  • Service containers, CodeQL, releases, deployments
2. Consume Workflows (20%)
  • Logs, debug mode, status badges, dependency graphs
  • Environments, approvals, branch protections
  • Download/upload artifacts, pass data between jobs
3. Author & Maintain Actions (25%)
  • Types: JS, Docker, Composite
  • action.yml: inputs, outputs, runs
  • Workflow commands: ::set-output, ::add-mask
4. Enterprise Management (15%)
  • Self-hosted runners, groups, policies
  • Org secrets, reusable workflows, starter workflows
  • Monitoring, troubleshooting runners
